This repository hosts multiple end-to-end data pipelines/engineering functions, classes and code. 
 - Data extraction: 
   - using SQL from Google Big Query
   - using gcloud storage or gsutil from GCP Buckets
   - using local files/data


 - Data curation and wrangling: perform complex data curation, wrangling and formatting to achieve insightful reports.

 - Report delivery: via auto generated notebooks, excel files or google cloud files.
